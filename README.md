# OJT_PROJECT
THIS IS OUR OJT PROJECT.
1. Project Title:
   Web Audio Player with Playlists & Waveform Scrubber

2. Description:
   This project is a modern web-based audio player that includes playlist management, waveform visualization, scrubbing, and smooth audio controls. It is built using HTML, CSS, JavaScript, and Wavesurfer.js to give users an interactive and clean listening experience.

3. Features:
   3.1 Play, pause, skip, and stop audio tracks
   3.2 Waveform visualization using Wavesurfer.js
   3.3 Playlist creation and switching
   3.4 Adjustable volume
   3.5 Responsive UI
   3.6 Smooth waveform scrubbing

4. Stretch Goals:
   4.1 Dark mode
   4.2 Local storage playlist saving
   4.3 Drag-and-drop audio upload
   4.4 Optional backend using Node.js

5. Tech Stack:
   5.1 HTML
   5.2 CSS
   5.3 JavaScript
   5.4 Wavesurfer.js
   5.5 Bootstrap
   5.6 Node.js (optional)

6. System Flow:
   6.1 Upload or select an audio file
   6.2 Waveform is rendered
   6.3 User can play, pause, and scrub through the track
   6.4 Playlist can be added or switched
   6.5 User can replay anytime

7. Testing:
   7.1 Unit testing for audio functions
   7.2 Manual UI testing
   7.3 Performance improvements for waveform rendering

8. Project Timeline:

   8.3 Week 3: Add playlist management and improve UI
   8.4 Week 4: Final testing, debugging, documentation, and demo
